# TheBank - Project Monorepo

This repository contains the full source code for "TheBank" application.

## Structure
- `/backend`: Java Spring Boot Application (API, Logic, Database)
- `/frontend`: Web Application (To be implemented)

## Getting Started

### Backend
Navigate to the `backend` directory to run the Spring Boot application.

```bash
cd backend
./gradlew bootRun
```

See `backend/README.md` for more details.
